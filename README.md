# Signals

Markov chain-based market signal generator for trading cryptocurrencies and equities.

## Overview

Signals analyzes extensive market datasets to build Markov chain models that identify trading signals. Currently focused on:

- **Bitcoin (BTC)** — initial target market
- **S&P 500** — planned expansion

## Roadmap

- [ ] Data pipeline for historical market data ingestion
- [ ] Markov chain model for state transitions and signal generation
- [ ] Backtesting framework to validate signals against historical data
- [ ] Signal dashboard / output
- [ ] Automated trade execution integration
